矩阵

定义的构造

A = 1:2:9（start:step:stop，开头，步长，结尾）
（相当于A = [1,3,5,7,9]）
B = [1, 2; 3, 4]
C = rempat(B, 2, 3) 

        C's graph:  1, 2, 1, 2, 1, 2;(第3个参数为3) 
                    3, 4, 3, 4, 3, 4; 
                    1, 2, 1, 2, 1, 2; 
                    3, 4, 3, 4, 3, 4; 
                    (第2个参数为2)

ones(row, col, d)（返回row行，col列，维度为d的全是1的矩阵）

矩阵的四则元素

A+B; A-B; A*B;这些都和数学上相同
A.*B（对应的项相乘）
A/B（相当于A乘以B的逆）
A./B（对应的项相除）

矩阵的下标

A = [1, 2, 3; 4, 5, 6; 7, 8, 9;]
A(2,3)（取第2行第3列的元素，也就是6）
A(3,:)（:表示所有，取第3行的所有元素，返回的是矩阵[7,8,9]）
A(:,3)（返回矩阵[3; 6; 9;]）
[m, n] = find(A>5) （m会接受到所有符合条件的行索引，n则是列索引）
运行后 m = [3; 3; 2; 3] n = [1; 2; 3; 3;]
可以发现A(m(i),n(i))的值是>5的，看起来是按列搜索的

前一节的内容

A' （返回A置换后的矩阵，可以用B=A'来接受返回值）
A(:) （返回的结果相当于矩阵[1 ; 4; 7; 2; 5; 8; 3; 6; 9]）
inv(A) （返回A的逆矩阵，定义：若矩阵A存在逆矩阵B，则A*B=B*A）
zeros(row, col, d)（row是行，col是列，d是维度；返回row行，col列，维度为d的全是0的矩阵）
rand(m, n)（返回m行n列的随机数，取值范围(0,1)，可以用矩阵接收；不填参数就返回一个随机数）
randn(m, n)（返回的是正态分布的随机数，均值为0，方差为1） randi([iMax,iMin], m, n)（返回的是随机整数，[iMax,iMin]是取值范围）
randi(iMax, m, n)（相当于randi([iMax,0], m, n)）
