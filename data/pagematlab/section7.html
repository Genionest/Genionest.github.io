求极限

matlab求极限是直接把数值代入函数去求极限值 

        >>clc, clear
        >>syms x;
        >>y = (1+x)^(1/x);
        >>limit(y, x, 0)  % 这个要输出，所以不加;
        
        ans = 
        exp(1)

求左极限 

        >>limit(y, x, 0, 'left')

求右极限 

        >>limit(y, x, 0, 'right')

求泰勒展开

        >>clc, clear;
        >>syms x;
        >>y = exp(-x);
        >>taylor(y, x, 2, 'order', 4) % 2表示展开点

        ans = 
        exp(-2)-exp(-2)*(x-2)+(exp(-2)*(x-2)^2)/2-(exp(-2)*(x-2)^3)/6

求不定积分

        >>clc, clear;
        >>syms x f;
        >>f = (3-x^2)^3;
        >>ans = int(f, x)

        ans = 
        -x^7/7+(9*x^5)/5-9*x^3+27*x

求定积分

        >>clc, clear;
        >>syms x;
        >>int(x*cos(x), 0, 1)

        ans = cos(1)+sin(1)-1

级数求和

        >>clc, clear;
        >>syms n k;
        >>f = 1/(k*(k+1));
        >>s = symsum(f, k, 1, n)

        s = 1-1/(n+1)
