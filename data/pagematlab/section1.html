变量

其他的定义都差不多，需要注意的是以下几个

字符串

a = 'abcd' 这里用的是单引号

注释

% 单行注释

矩阵

A = [1, 2, 3; 4, 5, 6; 7, 8, 9]

此矩阵位：
1, 2, 3
4, 5, 6
7, 8, 9
";"表示换行，空格和","都可以分开两个元素

矩阵方法

A' （返回A置换后的矩阵，可以用B=A'来接受返回值）
A(:) （返回的结果相当于矩阵[1 ; 4; 7; 2; 5; 8; 3; 6; 9]）
inv(A) （返回A的逆矩阵，定义：若矩阵A存在逆矩阵B，则A*B=B*A）
zeros(row, col, d)（row是行，col是列，d是维度；返回row行，col列，维度为d的全是0的矩阵）
rand(m, n)（返回m行n列的随机数，取值范围(0,1)，可以用矩阵接收；不填参数就返回一个随机数）
randn(m, n)（返回的是正态分布的随机数，均值为0，方差为1） randi([iMax,iMin], m, n)（返回的是随机整数，[iMax,iMin]是取值范围）
randi(iMax, m, n)（相当于randi([iMax,0], m, n)）

元胞数组

A = cell(2, 3)（返回一个2行3列的cell数组）
A = {1, 2, 3}（1x3的cell数组，定义就和c的数组类似）
数组A里的元素并不是1, 2, 3，而是3个cell，第一个cell里的元素是1；
第二个cell里的元素是2，cell像是某种数据结构，可以存放任意类型的变量；
matlab的数组索引是从1开始的，matlab使用索引是通过"()"和"{}"而非"[]"；
A(1)返回第一个cell，A{1}返回第一个cell里的元素1
A = {1, 2; 3, 4}（2x2的数组） A = { {1, 2}, {3, 4} }（1x2的数组，只是元素是数组）

结构体

A = struct('a', 1, 'b', {2})（hash表，写法是前一个键，后一个值，不能用;）
