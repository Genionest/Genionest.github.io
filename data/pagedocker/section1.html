添加镜像源

添加文件 sudo vim /etc/docker/daemon.json

{"registry-mirrors": [ "https://mirror.ccs.tencentyun.com", "https://docker.nju.edu.cn", "https://docker.mirrors.ustc.edu.cn", "https://docker.m.daocloud.io", "https://registry.docker-cn.com" ] }

重启docker服务 sudo systemctl daemon-reload sudo systemctl restart docker

查看镜像源是否配置成功 sudo docker info | grep Mirrors 如果能看到结果，则配置成功。

  1. 腾讯云 https://mirror.ccs.tencentyun.com
  2. 南京大学 https://docker.nju.edu.cn
  3. 中科大 https://docker.mirrors.ustc.edu.cn
  4. Daocloud镜像云 https://docker.m.daocloud.io
  5. Docker中国 https://registry.docker-cn.com
  6. 网易云 http://hub-mirror.c.163.com
