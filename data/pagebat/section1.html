变量设置

set设置变量 /a表示算数运算
set /a var = 1 + 2

设置字符串
set v = hello

获取变量的值需要在变量两端加上%
echo %var%

前面带有@，cmd框里就不会显示这行命令
@echo %var%

重定向(将输出内容添加到文件)

在当前目录下生成a.txt，并将"a"写入a.txt
echo "a" > a.txt
在当前目录下生成a.txt，并将"a"追加入a.txt
echo "a" >> a.txt
< 是将右边的内容写入左边的文件
<< 是将右边的内容追加入左边的文件

与或以及多命令执行

echo "a" && echo "b"
执行两个命令
echo "a" && ech
第一个执行，第二个报错
ech && echo "a"
第一个报错，第二个不执行
echo "a" || echo "b"
第一个执行，第二个不执行
ech || echo "b"
第一个报错，第二个执行
（有短路机制）

管道符号（前一个命令的输出等于后一个命令的输入）

显示当前目录下的所有文件
dir
显示当前目录下符合筛选条件（.txt结尾）的所有文件
dir命令的输出作为find命令的输入
dir | find ".txt"
