不使用第三个变量交换两个int变量的值

void swap_int(int *a, int *b){
    // 因为异或会记录两个数之间每个位上的异同情况，
    // 同时用异或后的数去异或原来其中一个数，又会返回出另一个数
    // 异或有这一性质：a ^ b = c; c ^ b = a; c ^ a = b;
    // 其实和 a + b = c; c - b = a; c - a = b;类似
    // 这里的巧妙之处在于 a = a + b之后a的值就不需要内存保存了
    // 而 b = a - b 之后b的值也不需要内存保存了
    // 按位异或最好，不会溢出
    *a = *a^*b;
    *b = *a^*b;
    *a = *a^*b;
}

其实这样做只是看上去只用了两个变量，实际在编译时还是会使用更多的变量来保存计算的值
直接就用三个变量交换其实更快，因为省去了计算的时间
