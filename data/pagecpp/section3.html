重载

--------------------
#include <iostream>
using namespace std;

class A{
public:
    A(){};
    void fn(){
        cout << 0;
    };
    void fn(int i){
        cout << i;
    };
};
--------------------

A类可以定义多个fn函数，但这些fn函数的参数表不能相同，
调用时也会根据参数表的不同选择对应参数表的函数进行调用，
此即为重载。

不能定义多个相同参数表，返回类型不同的同名函数

--------------------
// 对于之前定义的A类
class B: public A{
public:
    void fn(){
        cout << 1;
    };
};
--------------------
在其他oop语言里，子类可以重载父类定义的函数，
但在cpp里，如果fn是父类A定义的函数，B是A的子类，
B类里面有定义新的fn函数，B将失去其他所有A的fn函数，
只剩下B自己定义的这个fn函数，无论A是否定义了多个
fn函数

默认参数

---------------------
void fn(int i, int j=0);
void fn2(int i=0, int j);  /* 这是不合法的，默认参数后边的参数
也必须有默认参数 */
void fn3(int i=0, int j=0, int k=0);
---------------------

默认参数只能写一次，（对于.h文件保存原型，.cpp保存body的函数，默认
参数就只能写在.h文件的函数参数表里，.cpp文件里不能再写，因为只能写
一次。但也不能写在.cpp文件里，而不写在.h文件里，因为include的是.h
文件，就会导致找不到默认参数）

---------------------
// 对于之前定义的B类，如果想要调用A中的fn
B b;
b.A::fn();
b.A::fn(2);
---------------------
