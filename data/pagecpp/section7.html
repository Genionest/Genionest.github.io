static变量，static成员变量

    static变量
    static成员变量
static修饰一个变量，就是把局部变量变成全局变量，

全局变量只能在当前文件内使用，其他文件用extern也不行，
函数内的全局变量也只能通过函数调用去访问，
全局变量时在main函数之前进行初始化的，并且不规定次序，
所以不要让全局变量的初始化之间存在依赖。

static变量

static一个类的变量（这只是声明），需要在当前文件内再做定义，
可以在构造函数内赋值，并不能在构造函数的初始化列表里初始化，
因为static只能在定义的地方初始化（c++里要区别初始化与赋值）

static成员变量

---------------------
class A{
public:
    // A(ii):i(ii) {}; // 错误，只能在定义的地方初始化
    A(){ this->i = 10 };
    static int i;
    static void fn(){  
    /* 静态成员函数只能调用静态的成员变量，
    因为静态成员函数是不传入this的，不过作为函数，还是可以传参*/
        cout<<i;
    };
};

int A::i;  // 这里不能再加static，不然就重复了
// 也可以在这里进行赋值A::i=10;

A a;
cout<<a.i;  // 所有这个类的对象也都可以调用这个成员变量
cout<<A::i;  // 也可以直接获取
A::fn();  // a.fn()也是一样的
