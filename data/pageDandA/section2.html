RAM: Random Access Machine

以下每个操作只需要常数时间

寄存器顺序编号
R[0], R[1], R[2], ...

R[i] <- c (常数赋值)
R[i] <- R[j]
R[i] <- R[R[j]] (间接取址赋值)
R[R[i]] <- R[j]
R[i] <- R[i] + R[k]
IF R[i] = 0 GOTO 1 (条件判零，转向)
IF R[i] > 0 GOTO 1 (条件判正，转向)
GOTO 1 (转向)
STOP (停止)

循环不变式

初始化：循环的第一次迭代之前，它为真
保持：如果循环的某次迭代之前它为真，那么下次迭代之前它仍为真
终止：在循环终止时，不变式为我们提供一个有用的性质，该性质有助于证明算法时正确的
